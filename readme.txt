A powerful notification manager for WordPress. Collects alerts from WordPress Core, WooCommerce, and Contact Form 7 and sends them via Email, Telegram, and Slack.

== Description ==
Notification Hub is a central place to manage and send all your WordPress notifications.

Supports multiple channels:
* 📨 Email
* 💬 Telegram (Pro only)
* 🔔 Slack (Pro only)

Integrates with:
* WooCommerce – for new orders, status changes, and admin-created orders
* Contact Form 7 – for form submissions
* WordPress core – like new comments

== Features ==
* Unified notification dashboard
* WordPress Admin Bar alert badge
* Send test messages for all channels
* Settings page with tab-based UI
* Freemium model with Pro features

== Installation ==
1. Upload the plugin to `/wp-content/plugins/notification-hub/`.
2. Activate via the Plugins menu.
3. Go to “Notifications” in the admin menu.
4. Set up Email, Telegram, and Slack under Settings.
